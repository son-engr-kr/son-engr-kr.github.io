BCI Neural Visualization System

Precision Neuroscience BCI Hackathon - 1st Place ($1,000)

Real-time neural activity visualization and BCI array placement guidance system. Processes live data from a 1024-channel micro-ECoG array (32x32 grid) to help neurosurgeons optimize BCI array placement during surgery.

  • CNN-based denoising (ResNet U-Net with CBAM attention)
  • Kalman filter and EMA-based position tracking
  • Real-time heatmap smoothing with configurable web UI
  • 500 Hz sampling, ~20-50ms end-to-end latency
